# NoSQL Challenge: Social Network API
## Description
This project utilizes MongoDB, Express.js, and Insomnia to deomstrate the routes assoicated with creating users, thoughts, and reactions, similating a social network.

## Installation
To install this project, clone this repository and run npm start. Use Insomnia for testing routes.
